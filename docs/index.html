what

A simple and flexible module that sets the styling of a webpage to resemble a sequence of A4 pages along with some convenience methods to add text figures and more.

why

Generating PDFs programmatically is a nightmare. Not anymore: use HTML, CSS and Javascript to create beautiful books.

how

You can create books in the browser for development and then print them manually or just generate the pages in node and use a headless browser such as puppeteer to print it as PDF.

use in browser

<script type="text/javascript" src="boken.min.js"></script>
